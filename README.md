# Selenium Automation Framework

A scalable **Selenium Automation Framework** built using **Java**, **TestNG**, and the **Page Object Model (POM)**. It automates key user workflows of a sample e-commerce application.

---

## Tech Stack
- Java  
- Selenium WebDriver  
- TestNG  
- Maven  

---

## Features
- Login (valid & invalid scenarios)  
- Add products to cart  
- Checkout flow  
- Logout functionality  
- TestNG assertions  

---

## Framework Design
- Page Object Model (POM)  
- Modular and reusable structure  
- Easy to maintain and extend
